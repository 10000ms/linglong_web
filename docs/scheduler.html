Task Scheduler

Linglong Web integrates with aioclock to provide async task scheduling capabilities for background jobs and periodic tasks.

Why Task Scheduling?

  • Background Jobs - Run time-consuming tasks asynchronously
  • Periodic Tasks - Schedule recurring operations
  • Resource Efficiency - Offload non-critical work

Usage Example

from aioclock import Aioclock
from linglong_web import LinglongAppServer

app = Aioclock()

@app.cron("*/5 * * * *")
async def cleanup_old_data():
    # Run every 5 minutes
    await cleanup_service.run()
