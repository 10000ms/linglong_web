Utilities

Linglong Web provides various utility functions for common tasks.

Available Utilities

FunctionDescription
generate_uuid()Generate a UUID4 string
get_timestamp()Get current Unix timestamp
parse_datetime()Parse datetime from string
format_datetime()Format datetime to string
mask_sensitive()Mask sensitive data in strings

Usage Examples

# Generate UUID
uuid = generate_uuid()

# Get current timestamp
