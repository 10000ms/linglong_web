Quick Start

Quick Start

Get up and running with Linglong Web in minutes. This guide covers installation, basic setup, and your first API endpoint.

Installation

pip install linglong-web

Basic Setup

1

Create Configuration

Define your service configuration by extending LinglongConfigBase:

from linglong_web import LinglongConfigBase

class AppConfig(LinglongConfigBase):
    DEBUG = True
    SERVICE_NAME = "my-api"
    PGSQL_HOST = "localhost"
    REDIS_HOST = "localhost"
2

Initialize Resources

Set up database and cache connections:

from linglong_web import init_resources

await init_resources(
    db_config={"host": "localhost", "database": "mydb"},
    redis_config={"host": "localhost"}
)
3

Create API Endpoints

Use FastAPI's APIRouter with Linglong Web decorators:

from fastapi import APIRouter
from linglong_web.decorators import cacher, limiter

router = APIRouter()

@router.get("/users/{user_id}")
@cacher(ttl=60)
async def get_user(user_id: int):
    return {"id": user_id, "name": "John"}
4

Start Server

Initialize and start the Linglong Web server:

from linglong_web import LinglongAppServer

server = LinglongAppServer()
await server.initialize(
    service_name="my-api",
    router=router,
    config_dict={"dev": AppConfig}
)
await server.start(host="0.0.0.0", port=8080)

Complete Example

import asyncio
from fastapi import APIRouter
from linglong_web import LinglongAppServer, LinglongConfigBase, init_resources
from linglong_web.decorators import cacher

class Config(LinglongConfigBase):
    SERVICE_NAME = "demo-api"

async def main():
    await init_resources()
    
    router = APIRouter()
    
    @router.get("/hello")
    async def hello():
        return {"message": "Hello, World!"}
    
    server = LinglongAppServer()
    await server.initialize(
        service_name="demo-api",
        router=router,
        config_dict={"default": Config}
    )
    await server.start()

asyncio.run(main())
