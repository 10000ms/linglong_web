工具

工具函数

Linglong Web 提供各种常用工具函数。

可用工具

函数描述
generate_uuid()生成 UUID4 字符串
get_timestamp()获取当前 Unix 时间戳
parse_datetime()从字符串解析日期时间
format_datetime()格式化日期时间为字符串
mask_sensitive()掩码字符串中的敏感数据

使用示例

from linglong_web.utils import generate_uuid, get_timestamp

# 生成 UUID
uuid = generate_uuid()

# 获取当前时间戳
ts = get_timestamp()
