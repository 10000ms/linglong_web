Linglong Web

一个强大的异步 FastAPI 工具集，为生产级服务提供开箱即用的解决方案。 配置热重载、分布式缓存、限流、分布式锁等功能一应俱全。

为什么选择 Linglong Web？

构建生产级异步服务所需的一切

🔄

配置热重载

修改配置文件无需重启服务。非常适合对零停机有严格要求的生产环境。

🗄️

统一资源管理

从一个接口初始化和管理数据库（PostgreSQL/MySQL）、Redis、RabbitMQ、MongoDB 和 Celery。

⚡

分布式缓存

基于 Redis 的缓存装饰器，支持自动键生成和 TTL。在多个实例间扩展您的缓存。

🛡️

限流

基于 Redis 的限流，支持灵活的规则。在保持高可用性的同时防止滥用。

🔒

分布式锁

使用 Redis 实现集群级互斥锁。确保关键代码段在多个实例中只执行一次。

📅

任务调度

内置 aioclock 集成，支持定时任务和周期任务，内置错误处理和日志记录。

🔗

HTTP 客户端

基于 aiohttp 的封装，支持自动 request-id 透传和微服务间统一错误处理。

📊

DDL 管理

自动创建和迁移数据库表。在代码中直接管理您的数据库架构版本。

🎯

标准化响应

统一的 API 响应格式，提供成功/错误响应构建辅助函数。团队内部 API 设计一致。

与原生 FastAPI 对比

特性 纯 FastAPI Linglong Web
配置热重载 ❌ 需手动实现 ✅ 内置支持

分布式缓存 ❌ 需自行实现 ✅ @cacher 装饰器
限流 ❌ 需自行实现 ✅ @limiter 装饰器
分布式锁 ❌ 需自行实现 ✅ @cluster_lock

DDL 管理 ❌ Alembic/自行实现 ✅ AutoDDLManager
