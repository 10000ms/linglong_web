调度器

任务调度器

Linglong Web 与 aioclock 集成，提供异步任务调度功能，用于后台作业和周期性任务。

为什么需要任务调度？

  • 后台作业 - 异步运行耗时的任务
  • 周期性任务 - 调度重复操作
  • 资源效率 - 卸载非关键工作

from linglong_web import LinglongAppServer

app = Aioclock()
