Linglong Web

A powerful asynchronous FastAPI toolkit that provides out-of-the-box solutions for production-grade services. Configuration hot-reload, distributed caching, rate limiting, cluster locks, and more.

Get Started → View Examples

Why Choose Linglong Web?

Everything you need to build production-ready async services

🔄

Configuration Hot-Reload

Modify configuration files without restarting the service. Perfect for production environments where zero-downtime is critical.

🗄️

Unified Resource Management

Initialize and manage databases (PostgreSQL/MySQL), Redis, RabbitMQ, MongoDB, and Celery from a single interface.

⚡

Distributed Caching

Redis-based caching decorator with automatic key generation and TTL support. Scale your cache across multiple instances.

🛡️

Rate Limiting

Redis-backed rate limiting with flexible rules. Prevent abuse while maintaining high availability.

🔒

Distributed Locks

Cluster-wide mutex using Redis. Ensure critical sections are executed only once across multiple instances.

📅

Task Scheduling

Built-in integration with aioclock for cron jobs and periodic tasks with built-in error handling and logging.

🔗

HTTP Client

Wrapper around aiohttp with automatic request-id propagation and unified error handling across microservices.

📊

DDL Management

Auto-create and migrate database tables. Version control your schema directly in code.

🎯

Standardized Responses

Unified API response format with success/error building helpers. Consistent API design across your team.
